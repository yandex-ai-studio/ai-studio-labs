# Лабораторная работа по AI Studio

В этой обзорной лабораторной работе по AI Studio показано:

* Работа с **Yandex AI Studio** через OpenAI-совместимый SDK
* Взаимодействие с LLM через **Responses API**
* Создание изображений с помощью **YandexART** и их улучшение через итеративный агентный цикл
* Использование **VLM (Vision Language Models)** и структурных ответов (Pydantic) для анализа изображений
* Подключение внешних инструментов по протоколу **MCP (Model Context Protocol)** (погода, arXiv)
* Встроенные инструменты AI Studio: **веб-поиск**, **интерпретатор кода**
* Создание агента для **Deep Research** с использованием **OpenAI Agents SDK**
* Построение **мультиагентной системы** с координатором и специализированными агентами
